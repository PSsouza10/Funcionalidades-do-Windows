BLOCO 1

Área de Trabalho e Barra de Tarefas

~15 min

1.1 — Personalizar o Papel de Parede

  1. Clique com o botão direito na Área de Trabalho.
  2. Selecione "Personalizar".
  3. Escolha um novo plano de fundo (cor sólida ou imagem).
  4. Volte e altere para uma cor sólida de sua preferência.

1.2 — Organizar Ícones

  1. Clique com o botão direito na Área de Trabalho → Exibir.
  2. Altere o tamanho dos ícones para "Ícones grandes".
  3. Depois mude para "Ícones pequenos".
  4. Ative "Organizar ícones automaticamente" e veja o resultado.
  5. Desative e arraste os ícones manualmente para posições diferentes.

1.3 — Barra de Tarefas

  1. Clique com o botão direito na Barra de Tarefas → Configurações da barra de tarefas.
  2. Ative e desative a opção "Ocultar automaticamente a barra de tarefas".
  3. Fixe o aplicativo Calculadora na barra de tarefas (pesquise e clique com botão direito → "Fixar").
  4. Remova um ícone fixado (botão direito → "Desafixar da barra de tarefas").
BLOCO 2

Explorador de Arquivos

~30 min

2.1 — Criar Estrutura de Pastas

  1. Abra o Explorador de Arquivos (tecla Windows + E).
  2. Navegue até Documentos.
  3. Crie uma pasta chamada "Aula_Windows".
  4. Dentro dela, crie 3 subpastas: "Textos", "Imagens" e "Backup".
  5. Dentro de "Textos", crie mais uma pasta chamada "Rascunhos".

2.2 — Criar e Manipular Arquivos

  1. Dentro da pasta "Textos", clique com botão direito → Novo → Documento de Texto.
  2. Nomeie o arquivo como "meu_primeiro_arquivo.txt".
  3. Abra-o, digite seu nome completo e a data de hoje. Salve e feche.
  4. Crie mais 2 arquivos de texto: "notas.txt" e "lembrete.txt".
  5. Copie "notas.txt" para a pasta "Backup" (Ctrl+C → Ctrl+V).
  6. Mova "lembrete.txt" para a pasta "Rascunhos" (Ctrl+X → Ctrl+V).
  7. Renomeie "meu_primeiro_arquivo.txt" para "apresentacao.txt" (F2).

2.3 — Visualização e Busca

  1. Na pasta "Aula_Windows", alterne entre os modos de exibição: Lista, Detalhes, Ícones grandes.
  2. No modo "Detalhes", clique em "Nome" para ordenar alfabeticamente.
  3. Use a barra de busca (canto superior direito) e pesquise "notas".
  4. Observe que o arquivo aparece tanto em "Textos" quanto em "Backup".

2.4 — Lixeira

  1. Delete o arquivo "notas.txt" da pasta "Textos" (tecla Delete).
  2. Abra a Lixeira na Área de Trabalho.
  3. Encontre o arquivo e clique em "Restaurar".
  4. Verifique que ele voltou para a pasta original.
  5. Delete novamente e agora use Shift + Delete — observe a mensagem de exclusão permanente.

2.5 — Criar Documentos Word e Excel

  1. Abra Microsoft Word (procure no Menu Iniciar).
  2. Digite: "Relatório da Aula de Windows" como título.
  3. Abaixo, escreva um parágrafo sobre o que aprendeu na aula (mínimo 5 linhas).
  4. Salve como "relatorio_aula.docx" na pasta "Textos".
  5. Abra Microsoft Excel.
  6. Crie uma pequena tabela com 3 colunas: Atalho | Função | Testei
  7. Preencha com 5 atalhos que você praticou (ex: Win+E, Alt+Tab, etc).
  8. Salve como "atalhos_windows.xlsx" na pasta "Textos".

2.6 — Tirar Foto com Câmera (ou Usar Imagem Existente)

  1. Se tiver câmera, abra o aplicativo Câmera (Menu Iniciar).
  2. Tire uma foto de sua mesa ou local de trabalho.
  3. Salve como "foto_workspace.jpg" na pasta "Imagens".
  4. OU: Procure uma imagem no computador (Windows Spotlight, papel de parede, etc) e copie para "Imagens".
  5. Renomeie a imagem para "minha_imagem.jpg".
BLOCO 3

Gerenciamento de Janelas e Multitarefa

~15 min

3.1 — Snap e Redimensionamento

  1. Abra 3 aplicativos: Bloco de Notas, Calculadora e Explorador de Arquivos.
  2. Use Windows + Seta Esquerda para encaixar o Bloco de Notas na metade esquerda.
  3. Use Windows + Seta Direita para encaixar o Explorador na metade direita.
  4. Arraste a barra divisória entre eles para redimensionar.
  5. Use Windows + Seta Cima para maximizar o Bloco de Notas.
  6. Use Windows + Seta Baixo para restaurar e depois minimizar.

3.2 — Alt+Tab e Área de Trabalho Virtual

  1. Com os 3 apps abertos, pressione Alt + Tab e alterne entre eles.
  2. Pressione Windows + Tab para ver a Visão de Tarefas.
  3. Clique em "Nova área de trabalho" (canto superior).
  4. Na nova área de trabalho, abra o Paint.
  5. Alterne entre as áreas de trabalho com Ctrl + Windows + Seta Esquerda/Direita.
  6. Feche a área de trabalho extra (Windows + Tab → X na área de trabalho).

3.3 — Atalhos Essenciais

  1. Pressione Windows + D para mostrar a área de trabalho (minimiza tudo).
  2. Pressione novamente para restaurar.
  3. Pressione Windows + L para bloquear a tela (faça login novamente).
  4. Pressione Ctrl + Shift + Esc para abrir o Gerenciador de Tarefas.
  5. Observe os processos em execução e feche o Gerenciador.
BLOCO 4

Configurações do Sistema

~15 min

4.1 — Tela e Acessibilidade

  1. Abra Configurações (Windows + I).
  2. Vá em Sistema → Tela.
  3. Observe a resolução atual. Altere a escala para 125% e veja o efeito.
  4. Volte para 100%.
  5. Vá em Acessibilidade → Tamanho do texto e aumente a barra. Aplique e observe.
  6. Restaure o tamanho original.

4.2 — Data, Hora e Idioma

  1. Vá em Hora e idioma → Data e hora.
  2. Desative "Definir hora automaticamente" e altere a hora manualmente.
  3. Reative "Definir hora automaticamente".
  4. Vá em Idioma e região e observe os idiomas instalados.

4.3 — Aplicativos Padrão e Inicialização

  1. Vá em Aplicativos → Aplicativos padrão.
  2. Veja qual é o navegador padrão atual.
  3. Vá em Aplicativos → Inicialização.
  4. Observe quais programas iniciam com o Windows e o impacto de cada um.
BLOCO 5

Ferramentas Integradas e Captura

~15 min

5.1 — Captura de Tela

  1. Pressione Print Screen — abra o Paint e cole (Ctrl+V). Salve na pasta "Imagens".
  2. Pressione Alt + Print Screen para capturar apenas a janela ativa. Cole no Paint.
  3. Pressione Windows + Shift + S para abrir a Ferramenta de Captura.
  4. Faça uma captura retangular de uma parte da tela.
  5. Faça uma captura em formato livre (desenho à mão).
  6. Salve ambas na pasta "Imagens".

5.2 — Calculadora e Bloco de Notas

  1. Abra a Calculadora e mude para modo Científica.
  2. Calcule: 25² (625), √144 (12) e 15% de 480 (72).
  3. Mude para modo Conversor e converta 5 km para milhas.
  4. Abra o Bloco de Notas e anote os resultados.
  5. No Bloco de Notas, use Ctrl+H (Substituir) para trocar "km" por "quilômetros".
  6. Salve como "calculos.txt" na pasta "Textos".
